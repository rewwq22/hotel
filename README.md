# hotel-service-krd.ru — Проект сайта ОТЕЛЬ-СЕРВИС ЮГ

Сайт компании **ОТЕЛЬ-СЕРВИС ЮГ** — производитель и оптовый поставщик постельного белья и текстиля для гостиниц, отелей, хостелов и санаториев. Краснодар.

## Важные правила

1. **НЕ ТРОГАТЬ** Яндекс.Метрику в `<head>` (ID 106791450)
2. **НЕ УДАЛЯТЬ** файл подтверждения Яндекс.Вебмастера в корне сайта
3. **НЕ УДАЛЯТЬ** `mail.php` на сервере
4. Все пути в HTML — **относительные** (не абсолютные от `/`)
5. Каждая новая страница должна содержать полный header/footer по шаблону из AGENT.md

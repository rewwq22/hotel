# РЕКОМЕНДАЦИИ ПО ВНЕДРЕНИЮ

## Приоритет 1 — запустить немедленно:
- Главная страница (кластер 1)
- Категории каталога по типу изделий (кластер 2)
- Категории по типу ткани (кластер 3)
- Сопутствующий текстиль (кластер 5)
- Региональные страницы под Краснодар и Сочи (кластер 8)

## Приоритет 2 — в течение 1-2 месяцев:
- Посадочные по типам заведений (кластер 4)
- Страницы услуг (кластер 7)
- Фильтры по характеристикам (кластер 6)
- По звёздности (кластер 11)
- Остальные региональные страницы (кластер 8)

## Приоритет 3 — регулярная работа:
- Информационные статьи в блог (кластер 9) — по 2-4 статьи в месяц
- Конкурентные запросы (кластер 10)

## SEO-рекомендации:
1. **Title** каждой страницы содержит основной ключ кластера
2. **H1** — точное вхождение основного ключа
3. **Description** — включает 1-2 ключа + УТП (от производителя, доставка, цены)
4. **Перелинковка** — каждая категория ссылается на смежные (ткани → изделия → заведения)
5. **Хлебные крошки** — для улучшения структуры
6. **Фильтры** — не создают дубли (canonical, noindex для комбинаций)
7. **Контент** — каждая страница каталога имеет SEO-текст 1500-3000 символов
8. **Статьи** — 3000-7000 символов, с таблицами сравнения и структурированными данными
